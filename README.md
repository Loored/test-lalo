# test-lalo
Configuration for Express, React, Redux and SCSS applications.

## Getting started

Clone repo
````
git clone https://github.com/Loored/test-lalo.git
````

Install dependencies
````
npm i
````

Start development server
````
npm run dev
````

### Production
Build for production
````
npm run build 
````
Start production server
````
npm run start
````
